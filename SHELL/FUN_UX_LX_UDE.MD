```tar``` tape archiver (Create, extract, or list files from a tar file)

```
tar -xvf <file_name>
```

```./configure``` A configure script is an executable script designed to aid in developing
a program to be run on a wide number of different computers. It matches the libraries
on the user's computer, with those required by the program before compiling it from its source code.

Package types

```RPM``` Red Hat Package

Install

```
rpmi -i <package_name.rpm>
```

Update

```
rpm - U <package_name.rpm>
```

```DEB``` Debian PKG used by Ubuntu

Section 4 - Shell Scripting Guide

### <h3 id='10'>10 - Introduction to Shell Scripting</h3>

[[Introduction to Shell Scripting.udemy](https://www.udemy.com/fundamentals-of-unix-and-linux-system-administration/learn/v4/t/lecture/4115944?start=15)]

Emacs shortcuts

```
CTRL-E: jump to the end of the current line
CTRL-A: move to the line beginning
CTRL-P: bring the last used command
CTRL-R: search in command history
```

Vi shortcuts

```
$: go to the end of the current line
0: jump to the start of the current line
k: brings the last command typed
/: searches the history for commands
```

3 channels are available to a given process

```
STDIN - input
STDOUT - output
STDERR - error message
```

list out all processes

```
ps -ef | grep <proc_name>
```

list out to the directory where the proc is running

```
ls -l /proc/<pid>/fd
```

create/update a file

```
echo "this is some text" > temp.txt
cat temp.txt

>>>
this is some text
>>>

echo "this is some other text" > temp.txt
cat temp.txt

>>>
this is some other text
>>>
```

Append a text

```
echo "this is appended text" >> temp.txt
cat temp.txt

>>>
this is some other text
this is appended text
>>>
```

redirecting some logs

```
useradd user1
su - user1
find / -name core > output.txt 2 > error.txt
cat output.txt

>>>
list of files....
>>>

cat error.txt

>>>
list of errors...
>>>
```

using filters for find something

```
ls -l | grep <file_name>
```
